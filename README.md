# BiPlay - Simple VIP Landing Page

심플한 VIP 랜딩 페이지

## 기능

- ✅ 전체 배경 동영상
- ✅ 배너 동영상
- ✅ 텍스트 추가
- ✅ 버튼 추가 (텍스트 + 링크)
- ✅ VIP 호출 버튼 (텔레그램)
- ✅ 단일 JSON 저장
- ✅ 저장 즉시 반영

## 실행

```bash
npm start
```

## 배포 (Railway)

1. GitHub에 push
2. Railway에서 자동 배포
3. 환경변수 설정:
   - `PORT`: 자동 설정됨

## 파일 구조

- `index.html` - 프론트엔드
- `admin.html` - 관리자 페이지
- `server.js` - 백엔드
- `data.json` - 데이터 파일
- `uploads/` - 업로드 파일 (Git 무시)
